# 🛍️ ShopSmart

A sleek and responsive e-commerce experience built for modern online shopping.  
ShopSmart delivers a clean shopping interface with dynamic products, smart search, and a functional cart system — all powered by a lightweight frontend stack.

## ✨ Highlights

- ⚡ Dynamic product loading from API
- 🔍 Real-time product search
- 🛒 Add & remove cart items
- 🔢 Live cart counter
- 📱 Fully responsive design
- 🎨 Modern UI with smooth interactions
- 🏷️ Product categories and promotional sections

## 🚀 Built With

**Frontend**
- HTML5
- CSS3
- JavaScript (ES6)

**API**
- Fake Store API
